JAVA环境安装配置教程

update
产品,介绍,工作流引擎,java,OA,activiti
0. 环境准备

提示：如果您是开发者，想必已经有了下面的开发环境，可以自动忽略这步操作。
本产品使用较新的java8的运行环境，使用了很多java的新特性。
当前配置教程使用window 7 64位，相应的安装程序都是 64位的

  • 下载安装文件
    JDK1.8(http://pan.baidu.com/s/1skWfo3z)
    tomcat8.0(http://pan.baidu.com/s/1eRAgYOy)
    mysql(http://pan.baidu.com/s/1eSK6Abg)
    大概包含以下这些文件

OK，材料已经准备完毕，接下来就可以进行环境的安装了！

详细的安装步骤请跳转到这里(http://www.jianshu.com/p/73611f690eac)

安装成功之后开始进行产品的部署和安装～

尝试登陆：打开浏览器，输入：http://localhost/phpmyadmin


看到这个画面就是安装成功

输入用户名：root
输入密码：password 默认密码就是password
