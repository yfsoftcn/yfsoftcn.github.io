产品说明

本系统基于 JAVA MYSQL设计开发，提供所有源码与数据库脚本。

经过多年的OA系统实施经验，团队研发了一套适用于中小型公司内部管理的工作流平台，无需开发即可根据实际业务需求自定义流程模板，自定义业务模型。完全可视化的界面操作即可完成。

主要包含:

  • 文档管理
  • 工作日/周/月报管理
  • 行政管理
  • 车辆管理
  • 办公耗材管理
  • 公告管理
  • 系统管理

工作流引擎操作简单，使用方便，且团队正在研发与之配套的APP，能够轻松在手机上实现远程办公，无缝集成！

一、系统平台优势

  • 采用了Spring + MyBatis + bootstrap +jQuery + CSS3 的WEB技术，兼容当前主流浏览器，跨平台，在大多数移动终端设备上也能够正常渲染。

为保证浏览质量，请使用chrome(谷歌浏览器),firefox(火狐浏览器),IE10等兼容HTML5的浏览器浏览。

  • 界面采用METRO设计风格，扁平化的设计理念，符合现代的审美观念。

摒弃以往公司系统的老土界面，提升公司的整体形象。且更容易被使用者接受，操作更加简洁和人性化。

  • 严格遵循MVC开发的标准，将视图与控制器严格拆分，不会在jsp页面中参杂JAVA代码。

系统的所有UI控件与JAVA源码都是分开的，用户可自定义UI或后台功能，结构和层次都很清晰，方便二次开发。

  • 系统为纯原生JAVA代码开发，未使用任何第三方的收费插件，购买后不存在其他版权和收费问题。

*警告：不允许未经许可在公共场合下(包括任何网络媒体)发布本系统。*

  • 本系统可根据客户需求进行二次开发，本店提供技术支持(酌情收取适当费用)。
